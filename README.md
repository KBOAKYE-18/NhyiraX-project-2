# Photo Upload Web App

A simple web application that allows users to upload images to a server. Built with **Node.js**, **Express**, **MongoDB**, and **EJS**, this app demonstrates basic file handling, form submission, and storage in a database.

---

## Features

- Upload images from your local machine.
- Images are stored securely in a MongoDB database.
- Simple and responsive web interface using EJS templates.
- Basic validation to ensure users upload files correctly.

---

## Tech Stack

- **Node.js** – server runtime environment
- **Express.js** – backend framework
- **MongoDB** – database to store images
- **Mongoose** – MongoDB ODM
- **EJS** – template engine for views
- **Multer** – middleware for handling file uploads
